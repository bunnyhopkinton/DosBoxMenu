# Description
DOS Menu for DOS Box terminal interface.  This give you an ASCII menu system to launch various games, programs, and Windows 3.11.

# Installation
1. Copy menu.bat to your DOSBox or DOSBox-X C directory.
2. Add "menu.bat" to the end of your autoexec.bat file.
